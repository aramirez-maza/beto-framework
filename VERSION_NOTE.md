# Version Numbering Note

BETO reached v4.5.0 through four rapid iterations between March 12–18, 2026.
Each version represents a distinct architectural layer:

| Version | Layer Added |
|---------|-------------|
| v4.2.0 | Public release — core protocol + executor |
| v4.3.0 | Operational Semantic Closure (OSC) |
| v4.4.0 | Execution Efficiency and Routing Layer |
| v4.5.0 | SQLite Persistence Layer |

These are cumulative layers on a stable protocol core, not breaking changes or instability indicators.
The 11-step protocol, epistemic states, and TRACE mechanism are unchanged since v4.2.
